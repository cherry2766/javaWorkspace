# Java Study

## 📌 목적
- 자바 기초 학습
- Eclipse + Git 연동 연습

## 📚 학습 범위 (진행 기준)
- 🔄 Java 기초: 변수, 연산자, 제어문, 반복문, 배열
- 🔄 객체지향 기초: 클래스, 객체, 상속
- ⏳ 컬렉션: List, Set, Map / 제네릭
- ⏳ 추가 학습: 예외처리, API, 스레드

## 🧠 배운 점 요약
- 배열은 참조 타입이라 복사 시 주의
- static은 객체와 무관한 로직에 적합

## ❗ 자주 막힌 부분
- 기능 구현 도중 개념은 이해했지만 실제 코드 설계에서 막힘
  - for문과 if문의 순서 고민
  - 로직 흐름이 자연스럽게 연결되지 않음
- Java util 라이브러리가 많아서 어떤 클래스/함수를 써야 할지 모를 때가 있음
- 함수 사용법을 모르면 구석기식으로 for문, if문만 사용하게 되어 코드가 길고 복잡해짐
